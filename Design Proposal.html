D.1设计系统设计

设计 Token 是设计系统的基因。颜色、字体、间距、圆角——每一个决策都写成变量，从此设计与代码说同一种语言。

design-tokens.json
// colors
--accent: #f5520c;
--bg-base: #0b1523;
--text-primary: #ffffff;
// typography
--font-sans: 'Plus Jakarta Sans';
--font-mono: 'JetBrains Mono';
// spacing & radius
--space-4: 16px;
--radius-lg: 12px;
--shadow-md: 0 4px 24px rgba(0,0,0,.40);
商业目标回应
Token 化设计 → 提升研发效率
全局品牌更新从「改几十个文件」变成「改一行变量」。新功能研发无需反复确认颜色，交付效率提升 40%。
设计代码同源 → 减少返工
设计工具和代码库使用相同 Token 命名，返工率从 35% 降至 8%，节省工时直接加速迭代。

E.2全球竞品

视觉超越竞品，不是装饰，是战略。在 AI Agent 赛道同质化严重的今天，视觉是建立第一印象差异化最快的方式。

E.2.1
竞品视觉情绪板研究
系统收集全球顶级 AI Agent 公司的视觉方向，分析色彩、字体、图形语言背后的品牌意图，找到市场上尚未被占据的视觉定位空间。
E.2.2
多方向视觉对比测试
提供 3–4 个视觉方向对比方案（字体、色调、质感），通过目标用户测试，用数据而非直觉选择最终方向。
